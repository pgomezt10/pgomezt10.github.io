Seleccionar por nombre de etiqueta

Este texto esta seleccionado por etiqueta

Para seleccionar por etiqueta solo tienes que nombrar la etiqueta dentro de style y poner lo que quieres que pase

Seleccionar por id(#)

Este texto esta seleccionado por id

Para seleccionar por id tienes que darle un nombre cuando vas a poner la etiqueta de escribir

Seleccionar por clase(.)

Este texto esta seleccionado por clase

Para seleccionar por clase haces lo mismo que seleccionando por id, solo que en vez de poner id= tienes que poner clase=
